# HeroLib Web UI

Modern web UI framework for HeroLib, built with **V** + **Bootstrap 5**.

## Quick Start

```v
import freeflowuniverse.herolib.web.ui

mut app := ui.new_app()
app.title = 'My HeroLib App'
app.start(port: 8080)!
```
